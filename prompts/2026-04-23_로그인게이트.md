# 2026-04-23 로그인 게이트 추가

## 작업 목적/배경
- `public/data.csv`가 인증 없이 공개 URL로 노출되는 문제 해결.
- Vercel Serverless Functions 기반 HMAC 쿠키 세션으로 간단한 비밀번호 로그인 게이트 도입.

## 사용한 프롬프트 (요약)
- ot-report (Vite + React 19) 프로젝트에 Vercel Serverless Functions + 프론트 게이트로 비밀번호 로그인을 추가.
- `/api/login`, `/api/logout`, `/api/verify`, `/api/data` 생성. `api/_auth.ts` 공용 HMAC 유틸.
- `public/data.csv` → `data/data.csv`로 이동해 공용 경로에서 제거.
- `src/AuthGate.tsx` 신규 컴포넌트가 `main.tsx`에서 App을 감싸고, 미인증 시 로그인 폼을 렌더.
- `App.tsx`의 `fetch('/data.csv')`를 `fetch('/api/data')`로 변경.
- `vercel.json`에 `functions.api/*.ts` 런타임 지정, `.env.example`에 `AUTH_SECRET`/`LOGIN_PASSWORD` 기재.

## 주요 결과 요약
- HMAC-SHA256 서명 `{exp}.{sig}` 토큰을 httpOnly/SameSite=Lax 쿠키로 발급(기본 8h).
- `data.csv`는 `api/data.ts`에서 인증 쿠키 검증 후에만 내려주며, Vercel 번들에 `includeFiles: 'data/**'`로 포함.
- 빌드/`tsc --noEmit` 모두 통과.
- 배포 전 사용자 작업: Vercel Project 환경변수에 `AUTH_SECRET`(32자+ 랜덤), `LOGIN_PASSWORD` 등록 필요.
